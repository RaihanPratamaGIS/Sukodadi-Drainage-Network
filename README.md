# Sukodadi Drainage Network Mapping

## Overview

This project presents the spatial distribution of the drainage network in Sukodadi Village using ArcGIS.

## Objectives

- Map existing drainage infrastructure
- Support village spatial planning
- Visualize drainage distribution

## Software

- ArcGIS
- Microsoft Excel

## Skills

- GIS Mapping
- Cartography
- Spatial Data Management
- Infrastructure Mapping
